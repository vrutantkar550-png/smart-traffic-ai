# 🚦 Smart Traffic Signal Control using AI

## 📌 Overview
This project uses AI and Computer Vision to control traffic signals dynamically based on vehicle density.

## 🛠 Tech Stack
- Python
- OpenCV
- AI Logic

## ▶️ How to Run
1. Install dependencies:
   pip install -r requirements.txt

2. Run:
   cd src
   python main.py

## 📊 Output
- Detect vehicles
- Adjust signal timing dynamically
